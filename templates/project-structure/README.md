# Project Template

This is a starter template for your design sprint project. Copy this entire folder to start your project.

## Quick Start

```bash
# Copy the template
cp -r templates/project-structure ~/Projects/my-project-name

# Navigate to your project
cd ~/Projects/my-project-name

# Initialize git
git init
git add .
git commit -m "Initial project structure"

# Customize CLAUDE.md with your project details
# Then start working!
```

## Folder Overview

| Folder | Purpose | When to Use |
|--------|---------|-------------|
| `research/` | Discovery phase | Days 1-3 |
| `design/` | Define & ideate | Days 3-5 |
| `prototype/` | Building | Days 6-9 |
| `testing/` | User feedback | Days 7-9 |
| `presentation/` | Final deliverables | Days 9-10 |
| `journal/` | Daily reflections | Every day |

## Key Files

- `CLAUDE.md` - Project context for AI tools (customize this!)
- `README.md` - This file (replace with your project description)
- `design/design-brief.md` - Your design brief (due Day 5)

## Tips

1. Keep files in the right folders
2. Commit to Git daily
3. Update CLAUDE.md as your project evolves
4. Write in your journal every day
