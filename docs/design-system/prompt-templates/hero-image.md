# Prompt Template: Hero Image

> Use this template for generating hero/banner images with AI image tools (Midjourney, DALL-E, Gemini).

---

## Base Prompt

```
Create a hero image for [CONTEXT/PURPOSE].

CONCEPT: [DESCRIBE THE THEME OR MESSAGE]

STYLE (MDP Brand):
- Aesthetic: Abstract, modern, professional
- Feel: Warm and approachable, not cold/clinical
- Inspiration: Scandinavian design, medical clarity, tech simplicity

COLORS (use these exact values):
- Primary: Terracotta #E07A5F (warm, human)
- Secondary: Teal #3D5A6C (technical, systems)
- Accent options:
  - Amber #E8A838 (energy, action)
  - Purple #7B68EE (innovation, AI)
  - Sage green #81B29A (growth, health)
- Background: [DARK #1A1A1A / CREAM #FAF7F2 / NAVY #3D405B]

VISUAL APPROACH:
- Abstract shapes, no literal imagery
- Subtle geometric patterns
- Organic flowing forms OR angular chevron motifs
- Layered depth through overlapping shapes
- No photos of people's faces
- No cliché medical imagery (stethoscopes, pills, etc.)

COMPOSITION:
- Aspect ratio: 16:9
- Leave space on [LEFT/RIGHT/CENTER] for text overlay
- Balanced but not perfectly symmetrical
- Create depth through layered shapes, not 3D effects

MOOD: [OPTIMISTIC / INNOVATIVE / COLLABORATIVE / FOCUSED]

DON'T:
- Use busy gradients or glossy effects
- Add text or typography
- Include recognizable faces
- Use stock photo clichés
- Make it feel cold or clinical
```

```
- Flowing neural network patterns
- Organic medical/biological motifs (subtle DNA helix, cell-like shapes)
- Geometric shapes suggesting collaboration
- Layered translucent elements

COMPOSITION:
- Aspect ratio: 16:9
- Leave space on left third for title text

DON'T include: Literal AI imagery (robots, chips), medical equipment, faces
```

## Theme Variations

### Dark/Dramatic
```
Background: Near-black #1A1A1A
Accent: Warm terracotta #E07A5F glowing elements
Feel: Focused, immersive, evening session
```

### Light/Energetic
```
Background: Cream #FAF7F2 or Energy #FFFBF5
Accents: Multiple colors (terracotta, amber, sage)
Feel: Workshop energy, morning freshness
```

### Technical/Systems
```
Background: Cool #F0F4F8
Primary: Teal #3D5A6C
Feel: Analytical, structured, systems thinking
```

### Innovation/AI
```
Background: Light purple #F8F7FC or navy #3D405B
Accent: Purple #7B68EE with terracotta highlights
Feel: Experimental, future-focused
```

## Quality Checklist

Before using generated image:
- [ ] Colors match MDP palette (no approximations)
- [ ] No unwanted text or artifacts
- [ ] Sufficient space for text overlay
- [ ] Appropriate for healthcare context
- [ ] No cliché imagery
- [ ] Resolution suitable for use (16:9, high quality)
