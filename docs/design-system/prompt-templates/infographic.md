# Prompt Template: Infographic

> Use this template for generating infographics with AI image tools (Nano Banana Pro, Midjourney, DALL-E).

---

## Base Prompt

```
Create a minimalist infographic for [TOPIC/PURPOSE].

CONTENT:
- Title: [TITLE]
- Key points: [LIST 2-4 ITEMS]
- Data/metrics: [IF APPLICABLE]

STYLE (MDP Brand):
- Aesthetic: Clean, professional healthcare meets startup
- Primary color: Terracotta #E07A5F (warm, human)
- Secondary color: Teal #3D5A6C (technical, systems)
- Accent: Amber #E8A838 (for calls to action)
- Background: White #FFFFFF or warm cream #FAF7F2
- Text: Near-black #1A1A1A

TYPOGRAPHY FEEL:
- Headers: Modern monospace style (like code)
- Body: Clean, generous letter-spacing
- All caps for labels/categories

VISUAL ELEMENTS:
- Chevrons (< >) as design motifs where appropriate
- Rounded corners (8px radius feel)
- Generous whitespace
- Subtle shadows (not harsh drop shadows)
- Icons: Simple, geometric, outlined style

DO:
- Use asymmetric layouts
- Keep text minimal and scannable
- Include the terracotta accent for emphasis
- Add subtle geometric patterns

DON'T:
- Use busy gradients or 3D effects
- Add decorative flourishes
- Use stock photo clich√©s
- Overcrowd with text
- Use pure black (#000000)

LAYOUT:
- [PORTRAIT/LANDSCAPE/SQUARE]
- Leave space for [SPECIFIC PLACEMENT NEEDS]

OUTPUT: Single infographic, clean edges, ready for [DIGITAL/PRINT]
```

## Example: Team Progress Summary

```
Create a minimalist infographic showing team progress for a healthcare design sprint.

CONTENT:
- Team name: Team Checkmate
- Sprint day: Day 5 of 10
- Current phase: Ideation
- Key accomplishments:
  - Completed 6 user interviews
  - Identified 3 key insights
  - Generated 50+ solution concepts

STYLE (MDP Brand):
- Aesthetic: Clean, professional healthcare meets startup
- Primary color: Terracotta #E07A5F (warm, human)
- Secondary color: Teal #3D5A6C (technical, systems)
- Background: Warm cream #FAF7F2
- Text: Near-black #1A1A1A

TYPOGRAPHY FEEL:
- Headers: Modern monospace style
- Body: Clean, generous letter-spacing

VISUAL ELEMENTS:
- Chevrons (< >) as progress indicators
- Rounded corners (8px)
- Generous whitespace
- Small "<mdp>" watermark in bottom corner

LAYOUT:
- Portrait orientation
- Team name prominent at top
- Visual progress bar/timeline
- Accomplishments as icon + text pairs

OUTPUT: Single infographic, clean edges, ready for digital sharing
```

## Variations

### Data-Heavy Infographic
Add to base prompt:
```
DATA VISUALIZATION:
- Use simple bar/circle charts
- Terracotta for primary data series
- Teal for secondary data series
- Include clear labels on all data points
```

### Process/Timeline Infographic
Add to base prompt:
```
TIMELINE ELEMENTS:
- Horizontal or vertical flow
- Numbered steps in circles
- Connecting lines between steps
- Chevron (>) between phases to show progression
```

### Comparison Infographic
Add to base prompt:
```
COMPARISON LAYOUT:
- Two-column or before/after structure
- Use color to differentiate (warm vs cool)
- Clear headers for each side
- Aligned elements for easy scanning
```

---

## Brand Reminder

Always include these MDP elements:
- `<mdp>` watermark (small, corner placement)
- Terracotta (#E07A5F) as primary accent
- Monospace typography feel
- Generous whitespace
- No busy patterns or decorations
