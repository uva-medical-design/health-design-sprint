## Troubleshooting

### Common Issues

**"Command not found" errors**
- Close and reopen your terminal after installation
- Make sure you completed the PATH configuration step

**Claude Code won't authenticate**
- Verify you received and accepted the Claude Team invitation
- Check that you're signing in with your @virginia.edu email

**GitHub CLI asks for password repeatedly**
- Run `gh auth login` again and select HTTPS
- Choose "Login with a web browser" option

### Getting Help

If you're stuck:
1. Check the troubleshooting section in your OS-specific guide
2. Search the error message online (often solved on Stack Overflow)
3. Email the course team—we'll help before Day 1
